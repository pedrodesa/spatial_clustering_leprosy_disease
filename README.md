## Objetivo
Identificar fatores associados à incidência de hanseníase nos municípios brasileiros 
e compreender os padrões dos agrupamentos de detecção e seus determinantes de saúde e socioeconômicos.

## Fontes dos Dados
Os dados utilizados foram acessados por canais públicos.

| Fonte | Descrição | Link de acesso |
